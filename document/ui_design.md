# UI 設計與風格指南 (UI Design & Style Guide)

本應用程式採用 **現代玻璃擬態 (Modern Glassmorphism)** 風格，結合 Material Design 3 規範，打造具質感且資訊清晰的天氣介面。

## 1. 視覺風格 (Visual Style)

### 配色方案 (Color Palette)
| 顏色角色 | 色碼 | 描述 |
| :--- | :--- | :--- |
| **背景 (Background)** | `#2C333A` -> `#1B2026` | 深藍灰色垂直漸層，營造沉浸感。 |
| **卡片背景 (Glass Surface)** | `#2C333A` (Alpha 30-50%) | 半透明玻璃質感，搭配模糊效果與細微邊框。 |
| **主要文字 (OnSurface)** | `#FFFFFF` | 純白，用於城市名、溫度、主要資訊。 |
| **次要文字 (Secondary)** | `#B0B0B0` | 淺灰，用於標籤、日期、次要資訊。 |
| **強調色 (Accent)** | `#D0D5CC` | 用於按鈕或特定圖示背景。 |

### 圖示系統 (Iconography)
全面採用 **Vector Icons (向量圖示)** 以確保在各種解析度下的清晰度。
- **功能圖示**: `Settings` (設定/返回), `Search` (搜尋), `DateRange` (預報)。
- **天氣圖示**: 使用動態著色的向量圖示 (太陽 `Yellow`, 雲 `BlueGrey`, 雨 `LightBlue`)。

## 2. 畫面結構 (Layout Structure)

### 啟動畫面 (`LaunchScreen`)
- **功能**: 應用程式入口與資料載入。
- **UI**: 
    - 中央顯示圓形載入指示器 (CircularProgressIndicator)。
    - 下方顯示提示文字：「資料載入中，請稍後...」。
    - 背景延續主題深色漸層。

### 城市搜尋 (`CityListScreen`)
- **頂部**: 搜尋輸入框 (TextField)，支援輸入 2 字元後自動搜尋 (0.5s Debounce)。
- **列表**: 
    - 顯示城市名稱 (優先顯示在地化名稱 `localName`，如 "臺北市")。
    - 副標題顯示國家代碼。
    - 點擊項目即切換為主城市並進入天氣頁面。

### 主天氣頁面 (`WeatherScreen`)
採用垂直捲動佈局，由上而下包含：

1.  **頂部導航欄 (Top Bar)**:
    - 左側: **設定/切換城市** 按鈕 (齒輪圖示)。
    - 中間: **城市名稱** (在地化名稱)。
    - 背景: 透明，融入整體視覺。

2.  **今日天氣摘要 (Current Weather)**:
    - 視覺焦點，置中顯示。
    - 超大字級溫度顯示 (例如 "24°")。
    - 天氣狀況文字 (例如 "多雲")。
    - 詳細資訊列：最高溫/最低溫、體感溫度。

3.  **每小時預報 (Hourly Forecast) - 水平捲動**:
    - **容器**: 玻璃擬態卡片。
    - **互動**: 支援手指水平滑動查看未來 24 小時/每 3 小時趨勢。
    - **項目**: 時間、天氣圖示、溫度。

4.  **每週預報 (Weekly Forecast) - 垂直列表**:
    - **容器**: 獨立的玻璃擬態卡片或各別項目卡片。
    - **資訊**: 星期/日期、天氣圖示、溫度範圍 (Min - Max)。

## 3. 字體排版 (Typography)
- **Display Large**: 64sp+，用於當前溫度。
- **Title Medium/Large**: 用於城市名稱、卡片標題。
- **Body Medium**: 用於一般資訊、預報列表。
- **Label Small**: 用於輔助說明文字。
